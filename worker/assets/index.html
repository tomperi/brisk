Drop a folder. Get a site.

Static hosting plus a zero-config backend: database, files, AI, realtime, identity. No keys, no pipelines, no permissions. Everything here is visible to everyone, on purpose.

  1. scaffold a folder (or bring any folder of HTML)

    brisk init my-site

  2. ship it

    brisk deploy my-site

  3. that's the whole pipeline. need a database, AI, realtime? one script tag.
